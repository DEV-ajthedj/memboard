# memboard
A custom development board aimed at maximizing the RAM of an RP2354A. I am doing this because I am currently developing a 3D graphics pipeline (very experimental) for microcontrollers, and an upgrade from the 264 KB Pico would be very nice for larger frambuffers and 3D objects. This is also why I didn't really need much flash on the board, so I just went with the RP2354A, which has 2MB of flash already built in. This project is meant for me to be able to really start developing and testing my pipeline more, without being limited by memory.

# Future Improvements
In the future, I hope to make firmware to make this microcontroller stackable. This could (hopefully) substantially increase the maximum compute it could generate.
